# 地铁故障处置助手 (TrainFaultAssistant)

基于 React Native (Expo) 的地铁电客车故障应急处置查询应用，完全离线运行。

## 功能特性

- ✅ **完全离线**：知识库本地存储，无需任何网络连接
- 💬 **智能对话界面**：输入故障现象，自动匹配处置步骤
- 🎤 **语音输入**：支持语音描述故障（iOS 原生语音识别）
- 📚 **知识库浏览**：按分类浏览所有故障条目
- 🔍 **关键词检索**：支持 TCMS 代码、旁路开关、限速等关键词
- 🚄 **限速提示**：自动显示旁路操作后的限速要求

## 支持的故障类型

- 🚇 车门故障
- 🛑 制动故障
- ⚡ 牵引故障
- 🌐 网络故障
- 🔑 钥匙故障
- ❄️ 空调系统故障
- 🔥 火灾报警
- 💻 LCU 故障
- 🎯 受电弓故障
- 🔌 辅助供电故障
- 📋 通用处置步骤

## 数据来源

基于 **Q/SZGY 01 G 05.03.033-2025**《苏州轨道交通2号线电客车故障应急处理指南》

## 技术栈

- **框架**: React Native + Expo SDK 52
- **导航**: expo-router (基于文件路由)
- **语音**: expo-speech (离线 TTS)
- **状态管理**: React hooks
- **匹配算法**: TF-IDF + 关键词权重（离线运行）

## 环境要求

- Node.js ≥ 18
- Xcode ≥ 15 (iOS 开发)
- Android Studio + SDK (Android 开发，仅当你需要打包 Android 时)
